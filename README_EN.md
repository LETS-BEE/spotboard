# Spotboard

Spotboard is a scoreboard system for ACM-ICPC style programming contests. This project is implemented using modern web technologies, specifically Nuxt 4 and Vue 3.

## Tech Stack
- **Framework:** Nuxt 4, Vue 3
- **State Management:** Pinia
- **Design:** Material Design 3 (Expressive)
- **Language:** TypeScript

## Getting Started

### Installation and Execution

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Production build:
   ```bash
   npm run build
   ```

## Key Features
- Real-time scoreboard updates
- Team rankings, problem solution status, penalty display
- Auto/Manual feed control
- Search functionality

For more details, please refer to `spotboard-ui/README.md`.
